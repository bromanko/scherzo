# Show priority indicators in tasks output

Add visual priority indicators to task lines so high-priority work stands out.

Display format options:
- `!!!` / `!!` / `!` / ` ` for critical/high/normal/low
- Or: `[crit]` `[high]` `[norm]` `[low]`
- Or: Color coding if terminal supports it

Example output:
```
=== Pending (8) ===
!! s-839d Refactor orchestrator to separate VCS operations
   s-2443 Split large checkpoint.gleam module
!  s-b2f5 Replace custom int_to_string with stdlib
```

Implementation:
- Modify `format_task_line()` in commands.gleam
- Add `format_priority()` helper function
- Sort tasks by priority within each status group (critical first)
