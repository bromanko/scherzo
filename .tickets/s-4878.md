# Preserve JSON parse error context in checkpoint.gleam

At checkpoint.gleam:302, JSON parsing errors discard the actual error reason, returning only 'Failed to parse checkpoint JSON'. This makes debugging checkpoint loading issues difficult.

## Acceptance Criteria

Include original JSON parse error in error message
Use string.inspect(e) or similar to capture error details
Test with malformed checkpoint JSON
