# Make orchestrator config values environment-configurable

At orchestrator.gleam:28-33, config values are hardcoded with no way to override: max_retries=3, max_continuations=5. Different projects may need different settings.

## Acceptance Criteria

Read config from environment variables with defaults:
- SCHERZO_MAX_RETRIES (default: 3)
- SCHERZO_MAX_CONTINUATIONS (default: 5)
Consider config file support (.scherzo/config.toml)
Document all tunable parameters
