# Create agent config stubs on init

Modify `src/scherzo/cli/init.gleam` to create `.scherzo/agents/task/CLAUDE.md` stub during initialization.

## Changes

1. Add helper function:
   ```gleam
   fn create_agent_stubs(config_dir: String) -> Result(List(String), InitError)
   ```

2. Create stub at `.scherzo/agents/task/CLAUDE.md`:
   ```markdown
   # Task Agent Instructions

   <!-- These instructions REPLACE defaults. Customize how the agent works. -->

   ## Project Context

   Describe your project here.

   ## Coding Standards

   - Your conventions
   - Framework patterns
   - Testing requirements
   ```

3. Call `create_agent_stubs()` in `do_initialize()` after writing config.toml

4. Add created files to output list

## Acceptance Criteria

- [ ] `scherzo init` creates `.scherzo/agents/task/CLAUDE.md`
- [ ] Stub contains helpful template with comments
- [ ] Created files shown in init output
- [ ] `--force` overwrites existing stub
