# Replace let assert with explicit error handling in event bus subscribe

In bus.gleam:45-46, 'let assert Ok(id)' can cause a panic if the receive times out, crashing the calling process. Change subscribe to return Result(Id, String) and handle the timeout explicitly instead of asserting.
